# CanonicalTockySeq 0.1.0

* **Initial Public Release**: Beta release of the CanonicalTockySeq framework for single-cell transcriptomic manifold analysis.
* **GradientTockySeq**: Implementation of piecewise Spherical Linear Interpolation (SLERP) for geodesic trajectory reconstruction.
* **Conical Geometry**: Introduction of the "Colimaçon" (Nautilus) coordinate system to decouple temporal progression from signaling intensity.
* **Tocky Integration**: Full support for Nr4a3-Tocky Fluorescent Timer anchored landmark populations (New, Persistent, Arrested) in the tumour context.
